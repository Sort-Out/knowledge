撤销
1、撤销修改/删除的工作区内容
   git checkout <file>
2、撤销add到暂存区
   git reset head file
3、撤销commit到分支
   git reset --hard head^
4、撤销删除文件/文件夹
   git reset head [被误删文件]
   git checkout <file>
   
回溯
1、回溯上一个版本
   git reset --hard head^
2、回溯到特定版本
   git reset --hard 版本id
   
查看
1、查看提交历史
   git log
2、查看命令历史
   git reflog
   
删除
1、删除已经提交的无用文件[没有提交到版本库]
   git rm file
   git commit -m "..."
 
github使用
1、添加远程连接
   git remote add origin git@github.com:xxccodesave/2020_07_10.git
2、修改连接远程github仓库
   git remote set-url origin git@github.com:xxccodesave/2020_07_10.git
3、推送
   git push -u origin master [第一次]
   git push origin master [本地远程建立连接后]
   作用：把本地内容推送到远程，并将本地master分支和远程master分支关联起来
4、拉取
   git clone git@github.com:xxccodesave/2020_07_10.git

分支使用
1、分支创建
   git checkout -b dev 
2、分支切换
   git checout master/git checkout dev 
3、分支内容合并
   git merge dev 
4、分支删除
   git branch -d dev 
5、分支查看[当前使用分支]
   git branch
 
其他
1、添加所有修改 
   git add .
2、修改github仓库名，提交方法
   git add->git commit->git push
   
换电脑后重新配置:
1、配置ssh
	ssh-keygen -t rsa -C "你的邮箱地址"
	找到公匙文件夹，复制公匙内容到github settings. [后面git clone就可以使用ssh通道]
2、自报家门
	git config --global user.name "Your Name"
	git config --global user.email "email@example.com"
3、新建git仓库
   git init ; 新建仓库  
4、将新建仓库和远程仓库连接
	git remote add origin "你的git地址"
5、拉取内容
    git pull origin master
6、之后就可以修改了 
	git add .
	git commit
	git push -u origin master [第一次]
